## 概述
{: id="20210505163604-9bqjti3" updated="20210505163607"}

“剪藏”是指将 Web 页面上的内容复制到思源中，并尽量保持其原始格式。
{: id="20210505163537-f04zpvo" updated="20210505163740"}

## 使用方式
{: id="20210505163741-w3p5lx5" updated="20210505163757"}

* {: id="20210505164206-jydol2d"}如果不需要将剪藏内容中包含的图片拉取到本地，可直接在浏览器中选择复制，然后在思源中进行粘贴
  {: id="20210505164206-6jpvjkl" updated="20210505164440"}
* {: id="20210505164209-g9651e2"}如果想要将剪藏内容中包含的图片拉取到本地，则需要使用[思源浏览器插件](https://github.com/siyuan-note/siyuan-chrome)。#注意#：拉取图片到本地是付费订阅特性
  {: id="20210505164209-s3gbivw" updated="20210505164913"}
